# Semantic Keyword Extraction

A semantic keyword extraction engine built with Sentence Transformers and embedding similarity.

## Overview

This project extracts the most relevant concepts from a text using semantic embeddings instead of traditional keyword matching.

## Features

- Semantic similarity search
- Sentence embeddings
- Finance-oriented keyword extraction
- Modular architecture
- NLP pipeline

## Technologies

- Python
- Sentence Transformers
- PyTorch
- NLP
- Embeddings

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m src.main
```
